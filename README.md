# Covid-19 App

Created for university class assignment.

Demo: http://covid.hammerten.com

# Installation

npm required

```
git clone
cd covid_app
npm install
```

# Development

The website can run locally by calling:

```
npm start
```

# Deployment

Generate static build:

```
npm react-scripts build
```

## IMPORTANT

Generated files are stored in `build/` directory.
