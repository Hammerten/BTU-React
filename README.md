# reqres Api App

Created for university class assignment.

Demo: https://stoic-albattani-01b2ee.netlify.app
API: https://reqres.in

# Installation

npm required

```
git clone
cd midterm
yarn
```

# Development

The website can run locally by calling:

```
yarn start
```

# Deployment

Generate static build:

```
yarn build
```

## IMPORTANT

Generated files are stored in `build/` directory.
